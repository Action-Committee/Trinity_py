WINDOWS USERS:
--------------
To run the Trinity Wallet on Windows:

1. Double-click TrinityWallet.exe
2. The wallet GUI will open
3. No Python installation required!

Note: Windows may show a security warning because the .exe is not signed.
This is normal for open-source applications. Click "More info" then 
"Run anyway" to proceed.

BUILDING FROM SOURCE:
---------------------
If you prefer to build from source instead of using the pre-compiled 
executable, see the trinity_wallet_py/ directory and BUILD.md for 
instructions.

REQUIREMENTS:
-------------
- Windows 7 or later (64-bit)
- Trinity daemon (trinityd) running with RPC enabled
- At least 50 MB free disk space

SECURITY:
---------
- Executables are built automatically by GitHub Actions

- You can verify builds by checking the workflow runs
- For maximum security, build from source yourself
